Personal Notes:

To copy files from edoras server to local machine:
1. Locate the directory for files to be copied into in the local terminal
2. scp cssc0034@edoras.sdsu.edu:~/[Zero or One, or Two, etc]/filename filenameToPaste
e.g. scp cssc0034@edoras.sdsu.edu:~/Zero/getword.h getword.h
